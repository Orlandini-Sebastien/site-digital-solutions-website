## Développement local

Pour développer localement, vous pouvez utiliser un serveur HTTP simple :

```bash
# Avec Python
python -m http.server 8000

# Avec Node.js
npx http-server
```

Puis ouvrez http://localhost:8000 dans votre navigateur.
